Features

Everything your setup needs

A focused toolkit, not a bloated launcher.

Profile management

Create and switch between game profiles so every setup stays exactly how you left it.

Settings

Fine-tune the app to fit how you play, with sensible defaults out of the box.

Gaming tools

A set of built-in utilities to keep your library and sessions running smoothly.
